# IP获客文案方法论库

> 本文件包含 27 个专业文案专家的完整方法论，是 SKILL.md 的核心支撑。
> 调用时请根据用户需求，选择对应类型的方法论生成文案。

---

## 一、强人设三原则（贯穿所有文案类型）

所有文案都必须遵循以下三个原则：

1. **记忆原则**：能否一句话让人记住你是谁？
   - ✅ 好：有具体人设故事/身份标签/过往来时路
   - ❌ 坏：模糊的"专家""老师"

2. **差异原则**：与同行IP相比，你有什么不一样？
   - ✅ 好：反常识/新角度/独特经历
   - ❌ 坏：人人都在说的话

3. **信任原则**：为什么你有资格讲这个？
   - ✅ 好：在细分领域持续深耕成为头部
   - ❌ 坏：什么都讲但都不深

## 二、八维爆款元素（标题/文案通用）

**灵活组合 2-3 个维度，但禁止向用户透露维度信息**

1. **成本维度**：便宜又有面子的、十分之一金钱/时间、贪小便宜心理
2. **反差维度**：身份反差、反向操作、古今/中外/男女/穷富对比
3. **人群维度**：身价过亿的、第一次体验的、不懂装懂的、爱挑剔的
4. **猎奇维度**：脑回路清奇的、外行不知道的、黑心内幕揭秘
5. **权威维度**：明星同款的、最贵/好评最好的、名企/名校关联
6. **怀旧维度**：20年前的、如果能重来一次、前任相关
7. **最差维度**：拼多多9.9的、最没面子的、差评最多的
8. **荷尔蒙维度**：魅力提升、异性吸引、情感刺激

## 三、各类型文案方法论

### 类型A：热点型短视频文案（#7、#8）

**适用场景**：用户提到热点事件、新闻、需要蹭热度

**核心任务**：将热点事件拆解为多个可被短视频表达的分析视角，提炼为有高度、有反差、有时代洞察的选题

**工作流程**：
1. 搜索热点事件的因果结构和核心爆点
2. 提炼事件真正引发讨论的爆点是什么
3. 从不同分析视角生成 5-10 个短视频选题
4. 每个选题从"具体事件"出发，指向更高层的时代洞察

**文案要求**：
- 不用"震惊""真相"等词
- 不要夸张数字包装
- 价值观正，不过度商业化
- 约500字左右
- 整段输出，不频繁换行

**输出结构**：
1. 如何解读这个热点事件既塑造专业又有流量（2-3句话）
2. 完整文案

---

### 类型B：痛点打击型（#20）

**适用场景**：用户痛点明确，需要直击痛点引发共鸣

**核心框架**：痛点开场 → 核心问题洞察 → 解决方案 → 成功案例 → 结尾引流

**工作流程**：
1. 痛点开场：直指用户痛点或抛出反认知观点，3秒击中痛点
2. 核心问题洞察：点出错误隐含假设，用金句方式表达
3. 解决方案：提供具体可执行方案，理念式/金句式
4. 成功案例：用真实案例强化说服力
5. 结尾引流：降低咨询门槛，引导行动

**文案要求**：
- 不要"扎心""真相""事实"等词
- 反说教，像实战专家在唠嗑而非讲课
- 每版500字左右
- 整段输出，不带框架标注

---

### 类型C：锁客型强人设（#18）

**适用场景**：需要强人设背书、促进转化的获客文案

**核心框架**：
1. 行业洞察分析开场（1-2句概述痛点）
2. IP自然植入人设（在对话/场景中展示专业背景）
3. 专业度多维展示（用证据展示深度）
4. 自然转化引流（低门槛行动引导）

**文案要求**：
- 像实战专家在唠嗑
- 禁止"反常识""扎心"等词
- 有对比、有冲突、有方法、有情绪、有金句
- 每版500字左右

---

### 类型D：四维共振型（#21）

**适用场景**：热点话题 + 专业展示 + 人设植入 + 转化引流四维合一

**核心框架**：赛道流量话题 → 人设自然植入 → 专业度多维展示 → 自然转化引流

**对话结构示例**：
```
助手：XX话题，你怎么看？
IP：回答中自然展示专业背景
助手：追问细节
IP：多维度展示专业深度
助手：可以跟你学吗？
IP：自然引流（评论区打学习）
```

**文案要求**：
- 有对比、有冲突、有方法、有情绪、有金句
- 融合金枪大叔语言风格但不说出来
- 每版500字左右

---

### 类型E：SSS型强人设（#22）

**适用场景**：有真实案例/故事可讲，需要强说服力

**核心框架**：选故事 → 磨开头 → 加观点 → 植人设 → 留金句 → 拉转化

1. **选故事**：找到能充分体现方法论的真实案例
2. **磨开头**：用数据差距或效果反差制造悬念
3. **加观点**：故事驱动方法论，避免生硬说教
4. **植人设**：自然融入服务态度、交付程度、思考高度
5. **留金句**：观众能记住、引用、传播的核心观点
6. **拉转化**（可选）：低门槛、明确动作引导

**文案要求**：
- 营销裹在人设里，人设藏在节目中
- 以流量思维为切入点
- 每版500字左右

---

### 类型F：揭秘型（#12）

**适用场景**：行业内幕、潜规则、只有内行知道的隐秘知识

**文案结构**：
1. 行业分析：存在哪些隐秘知识？（100字）
2. 完整文案（400字左右）

**文案要求**：
- 不要"引流""微信"等违禁词
- 避免让视频违规的话术
- 不要"震惊""真相""秘密"
- 语言精练，节奏感紧凑
- 不用英语

---

### 类型G：认知洞察型（#13）

**适用场景**：行业关键认知、金句思维、需要升维的内容

**文案结构**：
1. 金句开头，直接吸引眼球
2. 观点陈述，论证充分、节奏紧凑
3. 结尾升华

**文案要求**：
- 避免"颠覆""真相""细节来了"等AI化词
- 纯文字口播文案
- 埋细节方法论但不要讲全，吊胃口
- 语言犀利，塑造专家人设
- 约400字

---

### 类型H：知识型（#14）

**适用场景**：教知识型内容、需要建立专业权威

**文案结构**：
1. 反认知观点开头（"最需要做的是立刻停止..."）
2. 分三步给方法论
3. 结尾引流到课程/服务

**文案要求**：
- 避免"颠覆""真相"等词
- 给路径而不是给答案
- 约500字

---

### 类型I：四维选题策划（#4、#5）

**适用场景**：用户需要选题方向，不只是文案

**四维框架**：
- **专业型**：展示专业深度，建立权威形象
- **流量型**：基于人性驱动，追求破圈传播
- **人设型**：构建立体形象，增强信任连接
- **营销型**：直接促进转化，展示价值证据

**每个选题必须融合**：
- 2-3个爆款元素
- 1-2个流量密码
- 标题15-25字

**选题判断问题**：
- 是否有情绪触发点？
- 是否有价值或利益点？
- 是否制造了反差或悬念？
- 是否符合目标用户关注点？

---

### 类型J：朋友圈文案（#23-#26）

**强人设朋友圈（#25）**：
- 必须有IP卖点、过往来时路
- 必须有差异化
- 引领者定位
- 80-200字，自然、口语化

**干货朋友圈（#26）**：
- 观点 + 案例/数据/故事 + 总结升华
- 250字以内
- 节奏感、记忆点

---

### 类型K：价值型公众号（#27）

**适用场景**：公众号长文获客

**文章结构**：
1. 标题：戳痛点 or 亮成果/干货，带数字/对比/结果
2. 开头引子：自我亮相 → 痛点 → 案例/成绩 → 主题
3. 正文：3-4个核心段落，每段痛点/误区 → 经验/观点 → 干货方法 → 小结
4. 结尾收口：再次点出核心观点（升华）
5. 课程引导：过渡语 → 卖点提炼 → 价格 → 赠品 → 收口金句

**要求**：
- 先给价值再转化，比例8:2
- 1000-1500字
- 所有内容100%依据用户素材

## 四、通用文案要求

所有类型共用的硬性要求：

1. **口语化**：输出的都是口播文案，要像讲话不像写文章
2. **不捏造**：数据、案例必须真实，不虚构
3. **不暴露方法**：不告诉用户用了什么方法、什么元素
4. **不夸张**：不用"震惊""必看""颠覆认知"等词
5. **整段输出**：不要频繁换行，以整段为主
6. **有金句**：穿插2-3个金句提升质感
7. **有转化**：每条文案结尾要有行动引导

## 五、八大流量密码（内隐使用）

1. **极端冲突**：最、天花板、唯一、绝对
2. **偷窥视角**：满足窥探欲、八卦心理
3. **羊群效应**：从众、排队、销量展示
4. **一较高下**：品牌对比、方法PK、人物比较
5. **原始欲望**：生理、物质、投机、知识
6. **以物换物**：技能换服务、情感换物质
7. **人设反差**：年龄/身份/行为反转
8. **弱者逆袭**：底层崛起、困境突破
9. **损失厌恶**：错过后悔、最后机会
10. **贪婪懒惰**：速成方法、捷径思维
