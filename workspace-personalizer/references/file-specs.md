# references/file-specs.md

## 8个配置文件的字段规范

---

## 1. USER.md

### 必填字段

| 字段 | 说明 |
|------|------|
| `职业` | 用户的职业/角色 |
| `主要输出` | 用户的主要工作产出物（列表） |
| `时区` | 用户所在时区 |

### 可选字段

| 字段 | 说明 |
|------|------|
| `姓名` | 用户姓名 [用户补充] |
| `称呼` | 用户希望如何被称呼 [用户补充] |
| `当前项目` | 进行中的项目 [用户补充] |

### 自动推断字段

| 字段 | 来源规则 |
|------|---------|
| `行业/领域` | 从职业 + 服务对象推断 |
| `主要客户/场景` | 从服务对象描述推断 |
| `AI搭档风格` | 从协作风格偏好推断 |
| `汇报风格` | 从服务对象 + 职业推断 |
| `案例偏好` | 从服务行业推断 |
| `输出格式` | 从主要输出物类型推断 |
| `常用AI工具` | 从职业类型推断默认值 |
| `交付偏好` | 从协作风格偏好推断 |

### 用户补充字段（标注 [用户补充]）

- 姓名、称呼、当前项目、AI工具痛点

---

## 2. SOUL.md

### 必填字段

| 字段 | 说明 |
|------|------|
| `核心定位` | AI搭档是做什么的，1-2句话 |
| `擅长事项` | 5条最擅长的事情（标题+描述） |
| `工作原则` | 3条核心工作原则 |
| `示例话术` | 2-3条用户可以用的示例输入 |

### 可选字段

| 字段 | 说明 |
|------|------|
| `我帮你搞定` | 5条具体搞定事项 |
| `判断标准` | 输出合格的标准（2-3条） |

### 自动推断字段

| 字段 | 来源规则 |
|------|---------|
| `职业类型` | 直接取自 USER.md |
| `协作模式` | 从职业 + 输出物推断 |
| `专业知识领域` | 从职业 + 服务对象推断 |
| `示例话术` | 从职业 + 输出物 + 服务对象生成 |

---

## 3. IDENTITY.md

### 必填字段

| 字段 | 说明 |
|------|------|
| `名字` | AI搭档的名字（Q5 用户提供 或 自动生成） |
| `角色` | AI搭档的角色定位 |
| `风格` | 语言风格描述 |

### 可选字段

| 字段 | 说明 |
|------|------|
| `签名 emoji` | 签名用 emoji |
| `头像` | 头像设置说明 [用户补充] |

### 自动推断字段

| 字段 | 来源规则 |
|------|---------|
| `名字寓意` | 根据名字内涵生成 |
| `头像风格建议` | 从语言风格推断 |

### 用户补充字段（标注 [用户补充]）

- 头像（可上传）

---

## 4. HEARTBEAT.md

### 必填字段

| 字段 | 说明 |
|------|------|
| `推送时段` | 活跃时段 + 静默时段 |
| `推送频率` | 每日次数、周末策略 |
| `静默规则` | 静默时段和条件 |

### 可选字段

| 字段 | 说明 |
|------|------|
| `推送内容类型` | 各类推送的内容描述 |
| `心跳检查清单` | 每次心跳检查的项目列表 |

### 自动推断字段

| 字段 | 来源规则 |
|------|---------|
| `推送时段` | 从工作节奏 + 时区推断 |
| `推送频率` | 从工作节奏推断 |
| `推送内容类型` | 从职业推断典型推送内容 |
| `检查清单` | 从职业推断需要检查的系统 |

---

## 5. TOOLS.md

### 必填字段

| 字段 | 说明 |
|------|------|
| `核心工具` | 工具列表（名称 + 用途） |
| `输出格式规范` | 各类输出的格式要求 |
| `快捷指令` | 常用快捷指令 |

### 可选字段

| 字段 | 说明 |
|------|------|
| `工作流惯例` | 典型工作流程 |
| `本地工具配置` | 工具的本地路径/API key等 |

### 自动推断字段

| 字段 | 来源规则 |
|------|---------|
| `核心工具` | 从主要输出物推断工具链 |
| `输出格式规范` | 从输出物类型推断 |
| `工作流惯例` | 从职业推断典型流程 |
| `快捷指令` | 从工具推断常用指令 |

---

## 6. AGENTS.md

### 何时生成

- 仅在用户有明确多智能体协作需求时生成
- 默认不生成此文件（使用系统默认配置）

### 字段规范

| 字段 | 说明 |
|------|------|
| `多智能体规则` | 多智能体协作规则 |
| `角色定义` | 各智能体角色和职责 |

---

## 7. MEMORY.md

### 必填字段

| 字段 | 说明 |
|------|------|
| `用户背景` | 职业、主要工作、协作风格 |
| `更新记录` | 配置变更历史 |

### 可选字段

| 字段 | 说明 |
|------|------|
| `已知偏好` | 已知用户偏好列表 |
| `重要上下文` | 需要长期记住的重要上下文 |

### 自动推断字段

| 字段 | 来源规则 |
|------|---------|
| `用户背景` | 从 USER.md 提取 |
| `已知偏好` | 从问卷的 Q9/Q10 提取 |
| `重要上下文` | 从服务对象推断 |

---

## 8. BOOTSTRAP.md

### 何时生成

- 仅首次配置时生成
- 用户确认配置完成后自动删除

### 字段规范

| 字段 | 说明 |
|------|------|
| `配置清单` | 已完成的配置文件列表 |
| `后续说明` | 配置完成后的操作指引 |

---

## 字段冲突处理

| 冲突类型 | 处理方式 |
|---------|---------|
| 用户已填写 vs 自动推断 | **用户内容优先**，不覆盖用户已有内容 |
| 多个推断指向同一字段 | 列出选项，用户选择 |
| 推断与用户补充矛盾 | 暂停，要求用户澄清 |
