# 全链路获客文案方法论库

> 本文件是 marketing-huoke-allinone 技能的核心支撑，
> 包含所有平台、所有文案类型的完整方法论。
> 调用时根据用户选择的平台和文案类型，调取对应章节生成文案。

---

## 通用原则（所有文案类型适用）

### 强人设三原则

1. **记忆原则**：能否一句话让人记住你是谁？
   - ✅ 有具体人设故事/身份标签/过往来时路
   - ❌ 模糊的"专家""老师"

2. **差异原则**：与同行IP相比，你有什么不一样？
   - ✅ 反常识/新角度/独特经历
   - ❌ 人人都在说的话

3. **信任原则**：为什么你有资格讲这个？
   - ✅ 在细分领域持续深耕成为头部
   - ❌ 什么都讲但都不深

### 八维爆款元素（灵活组合2-3个）

1. **成本**：便宜又有面子、十分之一时间/金钱
2. **反差**：身份反差、古今/中外/男女/穷富对比
3. **人群**：身价过亿的、第一次体验的
4. **猎奇**：脑回路清奇、外行不知道的内幕
5. **权威**：明星同款、名企关联
6. **怀旧**：20年前、如果能重来一次
7. **最差**：拼多多9.9、最没面子的
8. **荷尔蒙**：魅力提升、异性吸引

### 十大流量密码

1. 极端冲突（最、天花板、唯一）
2. 偷窥视角（满足窥探欲）
3. 羊群效应（从众、排队）
4. 一较高下（对比、PK）
5. 原始欲望（生理、物质）
6. 人设反差（年龄/身份反转）
7. 弱者逆袭（底层崛起）
8. 损失厌恶（错过后悔）
9. 贪婪懒惰（速成、捷径）
10. 以物换物（价值交换）

### 通用禁止项

- 禁止虚构数据、案例
- 禁止"震惊""颠覆认知""真相来了"等词
- 禁止暴露方法论（不告诉用户用了什么元素）
- 禁止说教模式，像实战专家在唠嗑而非讲课

## 第一部分：短视频文案

### 1. 痛点打击型（#20原版框架）

**适用场景**：用户痛点明确，需要直击共鸣

**文案结构**：
1. 痛点开场：直指痛点或抛出反认知观点，3秒击中
2. 核心问题洞察：用金句点出错误隐含假设
3. 解决方案：提供具体可执行方案，理念式/金句式
4. 成功案例（可选）：用真实案例强化说服力
5. 结尾引流：降低咨询门槛，引导行动

**文案要求**：
- 不要"扎心""真相"等词
- 像实战专家在唠嗑，不说教
- 约500字，整段输出
- 不带框架标注

### 2. 四维共振型（#21原版框架）

**适用场景**：热点话题+专业+人设+转化四维合一

**文案结构**：
1. 赛道流量话题切入（用热点/现象引发好奇）
2. 人设自然植入（在回答中展示专业背景）
3. 专业度多维展示（用证据展示深度）
4. 自然转化引流（低门槛行动引导）

**对话结构示例**：
```
助手：XX话题，你怎么看？
IP：回答中自然展示专业背景
助手：追问细节
IP：多维度展示专业深度
助手：可以跟你学吗？
IP：自然引流（评论区打学习）
```

### 3. SSS型强人设（#22原版框架）

**适用场景**：有真实案例/故事可讲

**文案结构**：
1. 选故事：找到能体现方法论的真实案例
2. 磨开头：用数据差距或效果反差制造悬念
3. 加观点：故事驱动方法论
4. 植人设：自然融入服务态度、交付程度
5. 留金句：观众能记住、引用的核心观点
6. 拉转化（可选）：低门槛、明确动作引导

### 4. 热点型（#7原版框架）

**适用场景**：追热点事件、蹭热度

**文案要求**：
- 不用"震惊""真相"等词
- 不要夸张数字包装
- 价值观正，不过度商业化
- 约500字，整段输出

**输出结构**：
1. 如何解读这个热点既塑造专业又有流量（2-3句话）
2. 完整文案

### 5. 揭秘型（#12原版框架）

**适用场景**：行业内幕、潜规则

**文案结构**：
1. 行业分析：存在哪些隐秘知识？（100字）
2. 完整文案（400字左右）

**禁止词**：引流、微信、震惊、真相

### 6. 认知洞察型（#13原版框架）

**适用场景**：行业关键认知、金句思维

**文案结构**：
1. 金句开头，直接吸引眼球
2. 观点陈述，论证充分、节奏紧凑
3. 结尾升华

### 7. 知识型（#14原版框架）

**适用场景**：教干货知识、建专业权威

**文案结构**：
1. 反认知观点开头
2. 分三步给方法论
3. 结尾引流到课程/服务

### 8. 锁客型强人设（#18原版框架）

**适用场景**：高客单、强人设背书

**文案结构**：
1. 行业洞察开场（1-2句概述痛点）
2. IP自然植入人设
3. 专业度多维展示
4. 自然转化引流

## 第二部分：公众号文案

### 9. 价值型公众号（#27原版框架）

**适用场景**：建立专业权威，自然转化

**文章结构**：
1. **标题**：戳痛点 or 亮成果/干货，带数字/对比/结果
2. **开头引子**：自我亮相 → 痛点 → 案例/成绩 → 主题
3. **正文**：3-4个核心段落，每段痛点/误区 → 经验/观点 → 干货方法 → 小结
4. **结尾收口**：再次点出核心观点（升华）
5. **课程引导**：过渡语 → 卖点提炼 → 价格 → 赠品 → 收口金句

**要求**：
- 先给价值再转化，比例8:2
- 1000-1500字
- 所有内容100%依据用户素材

### 10. 刘润式商业观察（公众号版）

**适用场景**：深度商业评论，B端获客

**文章结构**：
1. 现象引入（一个具体场景或事件）
2. 多角度分析（2-3个反常识视角）
3. 核心洞察（一句话升维结论）
4. 实战含义（对目标受众的价值）
5. 结尾引流（低门槛行动引导）

**风格**：克制、理性、有判断力，像见过很多案例的人在说真话

### 11. 故事型公众号

**适用场景**：有真实经历/创业故事可讲

**文章结构**：
1. 冲突开场（用最戏剧性的时刻切入）
2. 背景铺垫（为什么这件事重要）
3. 核心挑战（最大的困难/转折点）
4. 如何突破（方法论提炼）
5. 结果与感悟（升华）
6. 引流（低门槛咨询入口）

## 第三部分：小红书文案

### 12. 小绿书专业信任型（#小绿书原版框架）

**适用场景**：专业型IP在小红书建立信任

**内容结构**：
1. **开头**：真实困境切入，点出"想得大、落得小"的矛盾
2. **中段**：2-3个清晰概念，只讲作用和边界，不展开教学
3. **中后段**：强调真实流程、真实KPI，明确"不做什么"
4. **结尾**：反问式收口，引导决策者自我对照

**风格**：冷静、克制、专业，像顾问在复盘

### 13. 种草型小红书

**适用场景**：产品推广、真实体验分享

**内容结构**：
1. 真实体验开场（个人感受切入）
2. 产品/服务核心价值（1-2个亮点）
3. 适合人群（精准筛选）
4. 使用感受（具体细节）
5. 结尾（金句总结）

### 14. 干货型小红书

**适用场景**：实用技巧、方法分享

**内容结构**：
1. 痛点开场（目标用户的真实困扰）
2. 核心方法（1-2个关键步骤）
3. 注意事项（常见误区）
4. 适用边界（不是什么情况都适合）
5. 结尾互动（引导评论/收藏）

## 第四部分：朋友圈文案

### 15. 强人设朋友圈（#25原版框架）

**适用场景**：建立IP形象，持续曝光

**内容要求**：
- 必须有IP卖点、过往来时路
- 必须有差异化
- 引领者定位
- 80-200字，自然、口语化

**文案风格**：
- 不要说教
- 不要假嗨
- 不要流水账
- 像"清醒、见过很多案例的专业人士在自言自语"

### 16. 干货朋友圈（#26原版框架）

**适用场景**：输出观点，建立权威

**内容结构**：
1. 观点（一句话核心判断）
2. 案例/数据/故事支撑
3. 总结升华

**要求**：250字以内，节奏感、记忆点

### 17. 营销朋友圈

**适用场景**：促转化、推活动

**内容结构**：
1. 价值承诺（一句话说清楚能得到什么）
2. 核心卖点（1-2个亮点）
3. 行动引导（明确的下一步）
4. 紧迫感（限时/名额/门槛）

**要求**：约150字，有转化但不过度营销

### 18. 日常朋友圈

**适用场景**：维持活跃度、刷存在感

**内容要求**：
- 真实、自然
- 可结合工作日常
- 偶尔穿插观点
- 约100字

## 第五部分：其他类型

### 19. 直播脚本

**适用场景**：带货直播/沙龙公开课/招商直播

**脚本结构**：
1. 开场（3分钟）：建立信任，说明今天主题
2. 痛点激发（5分钟）：指出目标受众的核心困惑
3. 价值传递（10分钟）：分享干货/方法论/案例
4. 产品植入（5分钟）：自然引出产品/服务
5. 促单收尾（5分钟）：明确行动指引，限时/名额

**核心原则**：不能让观众感觉在看广告，要感觉在听分享

### 20. 私域成交文案

**适用场景**：高客单1v1转化

**文案类型**：
1. 破冰文案：引发好奇，自然开场
2. 价值传递：个性化展示你能解决什么
3. 促单文案：低门槛行动引导

**风格**：像朋友在给建议，不硬推

### 21. 课程招募文案

**适用场景**：招生、招募学员

**文案结构**：
1. 痛点切入（目标用户的真实困惑）
2. 课程价值（能解决什么问题）
3. 适合人群（精准筛选）
4. 特色亮点（为什么是你）
5. 行动引导（明确的报名方式）

## 第六部分：选题与策划

### 22. 四维选题策划（通用版）

**四维框架**：
- **专业型**：展示专业深度，建立权威形象
- **流量型**：基于人性驱动，追求破圈传播
- **人设型**：构建立体形象，增强信任连接
- **营销型**：直接促进转化，展示价值证据

**每个选题必须融合**：
- 2-3个爆款元素
- 1-2个流量密码
- 标题15-25字

### 23. 内容总策划（三层×九要素×27支点）

**吸引层（他为什么愿意停下来）**：
- Field（场域）：平台偏好、用户心智、获客路径
- Hook（钩子）：价值承诺、预期打破、情绪共鸣
- Audience（受众）：静态画像、动态场景、核心痛点

**信任层（我为什么要信你）**：
- Persona（人设）：身份定位、价值主张、人格魅力
- Value（价值）：显性价值、隐性价值、情绪价值
- Proof（背书）：权威背书、案例背书、自我背书

**转化层（我为什么现在要跟你走）**：
- Position（定位）：赛道卡位、优势组合、心智锚点
- Product（产品）：产品梯度、价值锚点、交付模式
- Action（行动）：明确指令、稀缺紧迫、风险承诺

### 24. 核心内容母题提炼

**三种母题**：
1. **痛点解决型**：解决当下最痛、最急的问题
2. **身份认同型**：帮助客户确认自己是谁
3. **价值成长型**：构建长期主义与组织价值

## 通用文案输出规范

### 标题规范

| 平台 | 标题长度 | 要求 |
|---|---|---|
| 短视频 | 15-25字 | 有情绪、有悬念、有价值 |
| 公众号 | 15-30字 | 带数字/对比/结果 |
| 小红书 | 15-20字 | 有共鸣、有干货感 |
| 朋友圈 | 不用标题 | 第一句话就是钩子 |

### 正文规范

- 短视频/口播：整段输出，不频繁换行，~500字
- 公众号：分段清晰，每段200-400字，1000-1500字
- 小红书：每段2-4行，500-800字
- 朋友圈：80-200字，口语化

### 结尾规范

- 所有文案结尾必须有行动引导
- 不要留"空"，要给用户一个明确的下一步
- 转化文案要有低门槛入口（评论/私信/扫码）
