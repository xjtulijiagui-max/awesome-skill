# Proactive Training AI - 课程设计智能体

> 一个能自动理解客户需求、匹配标准产品、生成定制课程大纲的智能系统。

## 核心能力

1. **需求解析** - 自动提取客户行业、受众、目标、时长等关键信息
2. **产品匹配** - 从你的标准课程库中匹配最适合的产品组合
3. **大纲生成** - 生成符合你风格的课程大纲（包含模块、工具、案例）
4. **质量检查** - 确保输出可直接交付

## 使用方法

### 直接输入客户需求

```
李老师好，这些内容您看看都可以讲吗？
AI通识类培训-银行技术人员-一天(2次)
1.AI时代变化、岗位转型、认知
2.大模型基础
3.互联网体系
4.金融范式
5.工具(提示词重点讲)
案例结合银行科技/互联网实际来讲
```

AI 自动识别：
- 行业：银行
- 受众：技术人员
- 时长：1天
- 重点：通识+工具
- 匹配产品：AI组织变革与管理创新 + AI场景创新

输出：完整的课程大纲 + PPT结构 + 案例清单

## 需求识别规则

### 关键词匹配

**行业关键词**：
- 银行/金融 → AI组织变革 + 银行案例
- 制造/半导体 → AI场景创新 + 制造案例
- 保险 → AI组织变革 + 保险案例
- 物业/服务 → AI场景创新 + 服务提效
- 设计/创意 → AI场景创新 + AIGC工具

**受众关键词**：
- 高管/总裁 → AI战略蓝图 + AI领导力
- 中层/经理 → AI组织变革 + AI领导力
- 技术人员 → AI场景创新 + 工具实操
- 全员/普通员工 → 西沙探险捕鱼

**目标关键词**：
- 战略/规划 → AI战略蓝图
- 组织/变革 → AI组织变革
- 场景/落地 → AI场景创新
- 领导力/管理 → AI领导力
- 意识/认知 → 西沙探险捕鱼

## 输出标准

所有生成的课程大纲必须包含：

1. **课程基本信息** - 名称、时长、对象、形式
2. **学习目标** - 3-5条可衡量的目标
3. **课程框架** - 模块→主题→内容三层结构
4. **工具清单** - 每个环节配套AI工具
5. **案例匹配** - 至少3个行业相关案例
6. **产出物** - 学员带走的具体成果
